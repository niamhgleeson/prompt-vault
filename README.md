# PromptVault

PromptVault is a Spring Boot web application for managing, storing, and submitting prompts to a simulated AI assistant.

## Technologies

- Java 21
- Spring Boot
- MySQL
- Thymeleaf
- Spring Data JPA

## Features

- User registration, login, and logout
- Admin login
- Admin user management
- Enable and disable users
- Prompt category management
- Policy keyword management
- Create, edit, delete, and view prompts
- Browse shared prompts
- Submit prompts to a simulated AI assistant
- Sensitive keyword warning
- Submission history
- Admin view of flagged prompts
